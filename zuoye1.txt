Git 与 GitHub 学习笔记
Git和Github是现代软件开发的核心工具。Git有仓库，提交，分支和远程仓库。我们所熟知的Github就是远程仓库的一种。仓库：是用来存储项目代码和版本历史的地方。提交：一次代码变更的记录，其中包含了提交者的信息，作者，时间。分支：独立的开发路线，一般默认是main或者master分支。
我们所熟知的github中不单单可以一个人进行项目的编辑，在有些时候我们需要团队的合作。我们要先Fork项目到我们自己的github账户中来，克隆自己的账号仓库，创建新分支，推送到自己的远程仓库，创建PULL REQUEST请求合并。最后我们要知道在每一次修改了代码后需要有提交说明，这可以让我们或者他人快速简洁的看出我们对代码修改的方向。
